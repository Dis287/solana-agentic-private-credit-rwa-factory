# Contributing to Solana Agentic Private Credit RWA Factory

Thank you for contributing!

## Quick Start
1. Apply to SDP waitlist: https://platform.solana.com
2. `git clone https://github.com/Dis287/solana-agentic-private-credit-rwa-factory.git`
3. `cd solana-agentic-private-credit-rwa-factory`
4. `pip install -r requirements.txt`
5. Run the swarm: `python agents/swarm.py`
